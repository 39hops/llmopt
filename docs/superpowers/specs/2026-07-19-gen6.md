# Gen-6: first territory birth, first predicted gate, grow-vs-rebirth

Provenance: the consolidation arc (Artin 2026-07-18/19). Gen-6 is
three experiments riding one corpus: (1) does new TERRITORY raise
the ceiling (the S_max thesis' first direct test), (2) does the
CALCULATOR predict a gate before the run (the equation's first
real forecast), (3) does GROWTH beat REBIRTH (dynamic capacity,
banked riff).

## Pre-registered predictions (publish BEFORE launch)

Calculator (v1 coefficients, k=21000, exposure update
c8: 319 -> ~14.5k):
- **L8 ceiling probe: ~12/24** (from 2/24) — the headline number
  the equation must hit (24*(1-exp(-14.5k/21k)) = 12.0).
- **Honest gate (L3-L7, 120): 69 +/- 2** — phylogeny says L8 is
  its own dialect (NCD), so cross-level transfer into the gate
  band is minimal; any big gate jump would REVISE the phylogeny.
- Growth arm keeps the champion's 69 base (function-preserving
  init) and matches the reborn arm's L8 gain; failure mode =
  rubble neurons (gradient hogging, the gen-5 reallocation
  lesson) which shows as L8 lagging the reborn arm at equal
  epochs.

## Arms (both fp32 on Mac, 3 epochs, identical data order)

- **A. REBORN-45M** (control): champion recipe verbatim (d=512,
  L12, ffn=2048, h8) on the gen-6 corpus. Fresh seed.
- **B. GROWN-45M->55M**: start FROM mathnative_45m_gen4_std.pt.
  Grow FFN only: ffn 2048 -> 2304 (+256/layer; ~+4.7M params,
  +9%, matched to corpus growth +10.7% — calculator-sized).
  New gate/up rows = template spray (near-orthogonal + 3% family

  preserving: identical outputs at step 0 — verify by gating the
  grown-untrained checkpoint == 69 exactly before training).
  scripts/grow_mathnative.py, one-time.
- C (follow-up slot, not this run): ternary-born gen-6 — after
  A/B verdict, the winning recipe gets its 1.58-bit twin.

## Metrics (both arms)

Honest 120-gate + L8 ceiling probe (24 fresh, seed band 69.1M) +
birth panel (CV/floor/R/norm/kurt) + committee confluence probe +
growth displacement portrait for B (ep0 vs final). Verdicts:
- Territory: reborn L8 probe >= ~10 => S_max thesis confirmed at
  predicted magnitude; gate stays ~69 => phylogeny confirmed.
- Calculator: |predicted - measured| on both numbers is THE
  equation scorecard, booked win or miss.
- Growth: B >= A on (gate, L8) at equal epochs => growth adopted
  as the standard capacity move; B << A on L8 => rubble diagnosis
  via panel (new-neuron norm/selectivity split out).
